Shopify Theme – Featured Products Section + Gulp/SCSS Workflow


This project is a customized Shopify theme based on Dawn, extended with:

1. A custom Featured Products section
2. Products hidden when already in the cart
3. Fully configured Gulp + SCSS workflow
4. Local development with Shopify CLI

Featured Products Section:

1. Select a collection in Theme Editor
2. Displays all products from the chosen collection
3. Responsive & mobile-first

Development Workflow:

1. SCSS compiled to assets/custom.css via Gulp
2. Auto-prefixing & minification
3. File watcher (gulp watch)
4. Shopify CLI Dev server (shopify theme dev)
5. Clean project structure (src/scss → assets/custom.css)
